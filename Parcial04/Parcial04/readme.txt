para poder ver la pagina web se descomprime el archivo dentro de nuestra
carpte de servidor ya sea xamp,wamp o laragon , por ejmplo en laragon es la carpeta www, ya una vez descomprimida la carpeta adentro de www, podemos pasar a probar el sistema

recordemos tener levantado el programa del servidor local
xamp,wamp o laragon, una ves levantado nos dirigimos a cmd, colocamos cd:(ruta 
donde esta la carpeta que descomprimimos) 

ya estando dentro de la carpeta que en este caso es parcial04 
procedemos a ejecutar el siguiente comando php -S localhost:8000, ya con esto podemos dar clic en el cmd y abrirá el navegador con el proyecto o también podremos irnos al navegador y escribir "http://localhost:8000/" 

y listo podremos visualizar todo
